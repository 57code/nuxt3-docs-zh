.gitignore 文件 Gitignore file #

通过 .gitignore 文件指定 git 应该忽略的故意取消跟踪的文件。在 git 文档 中了解更多信息。

我们建议使用一个 .gitignore 文件，该文件至少包含以下条目：

# Nuxt dev/build 输出
.output
.nuxt
# Node 依赖
node_modules
# 系统文件
*.log
