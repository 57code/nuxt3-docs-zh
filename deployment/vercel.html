Vercel #

如何将 Nuxt 部署到 Vercel 服务。

  • 支持无服务器构建
  • 部署时自动检测
  • 无需配置

设置 #

默认情况下，Vercel 会运行 package.json 中的 build 命令并且自动检测 .vercel_build_output/ 内构建的入口。

部署 #

和平常使用Vercel时一样，只需将其推送到 git 存储库中。
